# Story 1.2: Implement resilient Foreground Service with persistent notification

## Status
Approved

## Story
**As a** User,
**I want** to have the app run in background continuously,
**so that** it never stops monitoring my drowsiness.

## Acceptance Criteria
1. `VigilanceService` (Foreground Service) implemented in Kotlin.
2. Service type declared as `camera` and `specialUse` (Android 14 compliance).
3. Persistent notification with high priority and "Vigilance Active" message.
4. `PartialWakeLock` management to keep the CPU awake during monitoring.
5. Service lifecycle correctly handled (Start/Stop).
