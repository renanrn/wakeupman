# Story 2.3: Calibration UI for user baseline

## Status
Approved

## Story
**As a** User,
**I want** to calibrate my normal awake face,
**so that** the app accurately detects my specific drowsiness signs.

## Acceptance Criteria
1. UI screen to calculate user's baseline eye openness.
2. Save baseline to DataStore.

## 🤖 CodeRabbit Integration

>

## Tasks / Subtasks
- [ ] Task 1: Create `CalibrationScreen` with a camera preview.
- [ ] Task 2: Guide user to keep eyes open for 3 seconds.
- [ ] Task 3: Average the `eyeOpenProbability` over the period.
- [ ] Task 4: Save calculated baseline to Jetpack `DataStore`.

## Dev Notes
Normal eye openness varies, especially with glasses. If baseline is 0.7, the threshold for closed might be 0.4.

### Testing
- [ ] Verify calibration completes in 3 seconds.
- [ ] Verify baseline value is persisted across app restarts.
