# Story 3.4: Alert Stop UI (Big Red Button)

## Status
Done

## Story
**As a** User,
**I want** to tap a huge button to stop the alarm,
**so that** I can easily silence it when awake.

## Acceptance Criteria
1. Fullscreen `AlertActivity` launched from background.
2. Wakes up screen and unlocks if possible.
3. Giant red button to dismiss alarm.

## Tasks / Subtasks
- [x] Task 1: Create `AlertActivity` with window flags: `FLAG_KEEP_SCREEN_ON | FLAG_TURN_SCREEN_ON | FLAG_SHOW_WHEN_LOCKED`.
- [x] Task 2: Launch activity via Full-Screen Intent from the persistent notification.
- [x] Task 3: Compose UI with massive red touch target (Emergency Red).
- [x] Task 4: On click, send intent to `ServiceManager` to stop alerts and finish activity.

## Dev Notes
Activity must launch from background, requires full-screen intent on the notification to work reliably on Android 14. `AlertActivity` implements the "I'M AWAKE" button to allow users to easily silence the alarm.
