# Story 2.4: Detection Logic Engine (Threshold monitor)

## Status
Approved

## Story
**As a** System,
**I want** to trigger alerts based on rules,
**so that** the user is warned instantly.

## Acceptance Criteria
1. `DrowsinessEngine` collects data from AI.
2. Triggers 'Warning' or 'Emergency' states based on rules.
3. Rule: Eyes closed (prob < 40% of baseline) for > 2s = Emergency.
4. Rule: Head nod detected = Emergency.
