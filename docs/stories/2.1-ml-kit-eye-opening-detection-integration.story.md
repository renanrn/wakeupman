# Story 2.1: ML Kit Eye Opening detection integration

## Status
Approved

## Story
**As a** System,
**I want** to analyze eye open probability,
**so that** we can detect if the user is asleep.

## Acceptance Criteria
1. Google ML Kit Face Detection dependency added.
2. FaceDetector configured for speed (performance mode, classification on).
3. `Analyzer` from 1.3 passes `ImageProxy` to ML Kit.
4. `eyeOpenProbability` extracted and logged.
