# Story 1.4: Permission status dashboard

## Status
Approved

## Story
**As a** User,
**I want** to see and grant required permissions easily,
**so that** I can quickly start using the app.

## Acceptance Criteria
1. Screen built with Compose showing current status of Camera, Notifications, and Battery Optimization.
2. Buttons to request missing permissions or open Settings.
3. Navigation to Dashboard when all granted.
