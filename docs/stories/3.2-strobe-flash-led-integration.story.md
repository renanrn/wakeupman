# Story 3.2: Strobe Flash LED integration

## Status
Approved

## Story
**As a** User,
**I want** to see a bright flashing light,
**so that** it visually stimulates me to wake up.

## Acceptance Criteria
1. Access camera flash without interrupting CameraX.
2. Toggle flash rapidly (e.g., 100ms on, 100ms off) during Emergency.

## 🤖 CodeRabbit Integration
> **CodeRabbit Integration**: Disabled
>
> CodeRabbit CLI is not enabled in `core-config.yaml`.
> Quality validation will use manual review process only.
> To enable, set `coderabbit_integration.enabled: true` in core-config.yaml

## Tasks / Subtasks
- [ ] Task 1: Utilize CameraX `cameraControl.enableTorch(true/false)` to manage the flash.
- [ ] Task 2: Create a coroutine loop to toggle the state rapidly.
- [ ] Task 3: Ensure the loop stops when the emergency is dismissed.

## Dev Notes
Must ensure it doesn't conflict with CameraX image analysis. Using CameraX's built-in torch control is safer than raw CameraManager here.

### Testing
- [ ] Verify strobe effect is aggressive and consistent.
- [ ] Verify strobe stops immediately on dismissal.
